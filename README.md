## ASP_WebForm_test

> 簡單的購物車 + 部落格 demo

* 後端使用ASP.NET Web Forms + SQL SERVER EXPRESS
* 前端使用Bootstrap4.4.1

## 功能介紹
* 登入、登出會員系統
* 公眾、私人留言板(Gridview)功能
* 購物車取放功能、結帳、交易記錄查詢功能

## 使用方法

* 本機安裝SQL SERVER EXPRESS(版本2012以上)，建立資料庫asp_demo
* 於asp_demo資料庫執行：此專案目錄\ASP_WebForm_test\seed\seed.sql
* 使用VS編譯後測試
